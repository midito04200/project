# Association des Musulmans de Thonon (AMT) Website

This is the official website for the Association  of Moselems in Thonon-les-bains, built with React and Vite.

## Features

- Prayer times display
- Information about mosque activities
- New mosque construction project details
- Contact information
- Donation system integration

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- Headless UI
- Hero Icons

## Getting Started

1. Clone the repository
```bash
git clone [your-repo-url]
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```

## Project Structure

- `/src/components` - React components
- `/src/assets` - Static assets
- `/public` - Public files
- `/src/index.css` - Global styles
